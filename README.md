# Վերացական շարահյուսական ծառի գեներատոր

Ամեն մի տողը նկարագրում է մեկ էություն։ Ջավայի համար՝ interface, 
abstract class, final class: Եթե սլոթներ չունի, ապա interface է։
Եթե սլոթների ցուցակը դատարկ է, ապա abstract class է։ Եթե ժառանգներ 
չունի, ապա final class է։

Մուտաքային լեզուն ընտրված է այնպես, որ հնարավորություն լինի Ջավայից
բայց այլ լեզուներով տիպեր էլ գեներացնել։
